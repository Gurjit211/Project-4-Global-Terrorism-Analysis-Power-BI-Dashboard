# Global-Terrorism-Analysis-Power-BI-Dashboard

This project is a Power BI dashboard created to analyze global terrorism data.
It helps understand attack patterns, success rates, regions affected, and trends over time.

Overview

The main goal of this dashboard is to explore terrorism incidents across different countries, regions, attack types, targets, and weapons.
It also highlights the most affected years and the most active groups.

What the Dashboard Shows

Success rate by type of attack (Bombing, Armed Assault, Assassination, etc.)

Success rate by target (Civilians, Military, Police, Government, Business)

Success rate by weapon type (Explosives, Firearms, Incendiary, Melee)

Terrorism success rate by region

Country-wise terrorism activity shown on a world map

Most active and deadliest terrorist groups

Year-wise trend of total attacks

Maximum attacks recorded in a single year

Most commonly used weapon type

Key Insights

South Asia and the Middle East & North Africa have the highest number of successful attacks

Explosives are the most frequently used weapon

2014 recorded the highest number of attacks

Taliban is identified as the most deadly group in the dataset
